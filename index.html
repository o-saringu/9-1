関数の作成

サンプルコードの結果

サンプルコード

　　 void setup(){

　　 　println("Ready to roll!");

　　 　rollDice(20);

　　 　rollDice(20);

　　 　rollDice(6);

　　 　println("Finshed");

　　 }　

　　 　

　　 　void rollDice(int numSides){

　　 　　int d=1+int(random(numSides));

　　 　　println("Rolling..."+d);

　　 　}

説明文

　関数は自分で作ることができる。

　今回のプログラムではrollDice()という関数を作り、その関数の処理を[void　rollDice()]で行っている。

　rollDice()関数が読み込まれるたびに[void　rollDice()]の処理を毎回行う。

　

　このプログラムではrandom()関数を用いて、3つの異なる数字をコンソールに表示している。

　1、2個目のrollDiceでは1~20の乱数を表示し、3つ目は1~6の乱数を表示している。

　random()関数は浮動小数点数のため、整数を表示させるためには出力した値をint型にコンバートする必要がある。
